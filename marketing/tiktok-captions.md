## Posting playbook (quick)

- **Hook in 1.5s.** These open mid-image with the question already on screen — good. Don't add an intro card.
- **Caption first line = the hook**, because TikTok truncates. Put the link/CTA after.
- **One CTA:** `thesingleton.app` (free, no account, no email gate — say that; it lowers friction and people screenshot it).
- **Ask a question** at the end of the caption → comments → reach. Every top-tier caption above does this.
- **Pin a first comment** restating the question + link (same trick as the Show HN first comment).
- **Sound:** add trending or cinematic audio in-app (see warning #1). Non-negotiable.
- **Cadence:** 2–4/week, watch which hook lands, then double down on that flavor (identity vs. AI-twist vs. class vs. relationships).
- **Bilingual option:** the novel is EN/中文 and the genre is Douyin-native (机械纪元 style). If you want a 中文 set for Douyin/Bilibili or Chinese-TikTok, the hooks translate cleanly — ask and I'll write them.
