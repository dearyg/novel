# 中文短视频文案 — 《独子》(The Singleton)

生成于 2026-06-03。配合 `marketing/videos-vertical/ch01–ch20.mp4` 使用。
书名以 prose-zh 中既有的 **《独子》** 为准（英文版仍用 *The Singleton*）。

## 怎么用

- 这套文案是英文版 `tiktok-captions.md` 的中文平行版本。两种用法：
  1. **国内平台**（抖音 / 小红书 / B站 / 视频号）单独发中文。
  2. **TikTok 双语**：在英文文案下面接一段中文，覆盖海外华语观众（见文末「双语叠加」）。

## ⚠️ 发之前注意（和英文版相同 + 国内平台特有）

1. **视频没有音轨。** 抖音/TikTok 对无声视频限流很重。发布前在 App 内加一段背景音（热门音乐或电影感氛围乐），这是回报最高的一步。
2. **`ch15` 画面里有 AI 乱生成的「A24」广告牌**（A24 是真实片商）。想稳妥就把 ch15 往后排或放大避开那一帧。
3. **国内平台外链受限：**
   - 抖音正文一般不能放可点的外链 → 把 `thesingleton.app` 放在**主页简介**，正文写「主页直达 / 全文免费」。
   - `thesingleton.app` 在中国大陆可能打不开 → 如果要认真做国内，建议配一个**公众号 / 小程序 / 国内可访问的镜像**作为落点。
   - 话题格式：抖音用 `#话题#`，小红书用 `#话题`，B站单独填 tag。下面统一给关键词，按平台改格式。
4. **视频里的字是英文。** 给国内观众，最好把屏幕上的钩子换成中文重渲染（下方每条都给了中文屏幕钩子）。要的话我可以批量烧进视频。
